itálico / ênfase

tag i para itálico (forma)(não semantica)

tag 'em' para ênfase (semântica)

recomenda-se sempre utilizar as tag semanticas, pois o html5 é chamado de html semântico, então o que se espera é que as tags não semanticas entrem em desuso no futuro

teste usando as tags semânticas e não semanticas misturadas

'style' na head do site 'style'
