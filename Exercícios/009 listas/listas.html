listas

listas ordenadas= tag 'ol' (ordered list);
tag 'li' para cada iten (list item) cada iten deve estar dentro da tag 'li'

  1. dessa forma
  2. cada iten
  3. é numerado
  4. automaticamente.

OBS1: o fechamento da tag 'li' se tornou opcional na versão 5 do hmtl.

OBS2: a tag 'ol' tem parametros:

parâmetro 1: 'type' escolhe o tipo de lista, podendo ser em números, letras e algarismos romanos, se este parâmetro não for modificado a numeração será com números

parâmetro 2: 'start' escolhe qual será a primeira numeração lista, independete do tipo de lista a no parâmetro start deve ser usado número, pois ele é um parâmetro numérico.

listas não ordenadas= tag 'UL', quando a ordem da lista não é orbigatória.

parâmetros são, disk, circle e square

por padrão o parâmetro é o disk.
