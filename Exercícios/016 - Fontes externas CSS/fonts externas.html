Adicionando fontes externas:

Baixe a fonte que deseja, após isso extraia a pasta, geralmente aparece 2 arquivos, um otf e outro ttf

Comando '@font-face': Este comando faz vc 'criar' a fonte dentro do seu código, e você pode dar o nome que deseja para esta fonte

            @font-face {
            font-family: 'nome que deseja atribuir à fonte';
            src: url('caminho e nome do arquivo baixado') format(formato da letra baixada)'1ª letra', 
            url('caminho e nome do arquivo baixado') format(formato da letra baixada)'2ª letra'.
            }

EXEMPLO:

                
                    @font-face {
                    font-family: 'love';
                    src: url('love larry.otf') format(opentype), url('fonts/love larry.ttf') format(truetype)
                    }

os formatos das letras podem ser:

            - opentype (otf)
            - Truetype (ttf)
            - Embedded-opentype
            - truetype-aat (Apple Advanced Typography)
