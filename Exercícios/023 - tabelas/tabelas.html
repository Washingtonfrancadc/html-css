Hierarquia de Tabelas

table = tabela

table row tag 'tr' = linha de tabela

table header = cabeçalho de tabela

table data tag 'td'= dados de tabela

alinhamento vertical = 'vertical-allign' top, middle ou bottom

Efeito zebrado pseudo-classe comando = ':nth-child(2n)' (2n=de 2 em 2,/ 3n=de 3 em 3...)
linha impar=odd, limpar par=even

NOTA: código usado para aplicar em tabelas mas serve tbm para selecionar determinada linha da tabela para q sejam aplicadas qualquer configuração de estilo, como cor de fundo, tamanho da letra, decoração de teto...

OBS: table data fica dentro da table row

Efeito para colar cabeçalho no topo:

comando : 

        thead > tr > th {
        'position: sticky;
        top: -1px;
        Background-color: 'x'
        }
