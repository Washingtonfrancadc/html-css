'h1' título principal

'h1' título secundário

Atribuindo 'ids' às tags:

para atribuis 'ids' às tags usa-se o seguinte comando: <'h1 id="atibua o nome">, em vez de apenas '< h1>'

EXEMPLO:

< h1 id="principal">lerolerolero< /h1>

apareceria assim:

lerolerolero

lerolerolero

E nas config das CSS usa-se o comando 'h1#principal' para selecionar o título q qr modificar, caso tenha usado a 'id="principal" no código da html.

OBS.: Note q apenas um 'h1' está com a formatação diferente, pois foi atribuído apenas à ele o ID.

ID tem q ser único, em CSS representado pow '#', Class podem ter várias, em CSS representado por '.' (ponto)
Para selecionar as classes nas CSS, não é necessário colocar o nome do elemento, apenas o identificador da class: caso tenha um h1 com class 'inicio' e um h2 com class 'comeco' basta colocar o '.inicio' para selecionar o h1 com essa classificação, e '.comeco' para o h2 com essa classificação.
Pode ser atribuída mais de uma classificação para o mesmo elemento, e pode ser atribuída uma classe para um elemento q já tem ID
